Colour — spec §4.1

The accent is the only saturated colour on the site, and is reserved for interactive or signalling elements. Never decoration, never a large fill outside buttons.

Type scale — spec §4.2

Playfair Display 500 for display, Helvetica Neue system stack for body and UI. Sentence case throughout; uppercase is reserved for kickers. Resize the window — every display size below is fluid down to 360px.

Focus styles — spec §8

Tab through this page. Every interactive element takes a 2px --accent outline offset 2px. In the dark footer the outline switches to --paper, because accent on ink is far too low-contrast to serve as a focus indicator — spec §8 allows "or equivalent".
